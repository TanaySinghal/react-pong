Component que sea como el game engine renderer

- GameObjects

GameObjects have components

<GameEngine
  objects={objects}
  setting={settings}
/>

Delega las cosas a sus miembros:
  InputHandler - Revisa los inputs
    * start
    * end
  GameLoop - Corre los scripts de los objects
    * start
    * end
  Renderer - Renderiza los objects
    * render

Intervaler {
  int id;
  void start(int ms) {
    id = setInterval(run, ms);
  }
  abstract run();
  void end() {
    clearInterval(id);
  }
}
